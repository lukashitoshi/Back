# Desafio Técnico - Backend

Desenvolvimento do Backend de uma aplicação de um quadro Kanban

## Tecnologias

C# + ASP.NET Core + WebApi

## Rodando o Backend

Para rodar o Backend,

```console
> cd Back
> dotnet tool install --global dotnet-ef
> dotnet ef database update
> dotnet run
```

## Dificuldades

Mesmo após tentativas para implementar o docker-compose na aplicação, a sua realização não foi possível pelo desconhecimento e limite de tempo
A entrega fora do prazo se deu por complicações pessoais.
